# v0.1.8: Mermaid 图表可视化 + 欢迎语修复

## 方案对比

| | `@milkdown/plugin-diagram` | `@xz-summer/milkdown-mermaid` |
|---|---|---|
| 状态 | ❌ 已废弃 | ✅ 持续维护(v0.1.2, 2026年1月) |
| API | 停止支持 | 简洁：`editor.use(mermaidPlugin)` |
| 图表类型 | 全系列 | 全系列（flowchart/sequence/class/er/gantt/pie等） |
| 依赖 | 不维护 | 仅 `mermaid` 一个运行时依赖 |
| 配置 | - | 主题/占位文字/错误提示 都可配 |

**结论：采用 `@xz-summer/milkdown-mermaid`**

## 实现步骤

1. `npm install @xz-summer/milkdown-mermaid mermaid`
2. `editor.ts` 中添加 `import { mermaidPlugin } from '@xz-summer/milkdown-mermaid'`，`.use(mermaidPlugin)`
3. `base.css` 添加 mermaid 容器样式
4. 修复 `Welcome to ColaMD` → `Welcome to huasMD`

## 用户使用方式
````markdown
```mermaid
flowchart LR
  A[开始] --> B[处理] --> C[结束]
```
````
编辑器内实时渲染为 SVG 图表，导出 PDF/HTML 时也会保留。
